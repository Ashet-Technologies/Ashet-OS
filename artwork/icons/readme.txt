Web Mini icon set has been designed to illustrate web sites, blogs or web-oriented applications. The size of the icons is 16x16 which make them suitable for small toolbars. This small size permits also an easy integration in websites, blogs or forums. 
This set, which contains 237 icons, is part 1 of a larger set. Next parts will be released soon... Stay tuned. 

Icons are provided in formats PNG (32 bits - RGB/A), GIF (8 bits - 256 colors) and ICO (RGB/A). All the packs are available for download at this page: http://www.axialis.com/free/icons
